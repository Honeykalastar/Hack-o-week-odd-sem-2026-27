# Student Performance Analyzer

## Project Overview

The Student Performance Analyzer is a Python-based data analytics project developed to analyze student academic performance using Python, NumPy, Pandas, Matplotlib, and Seaborn.

This project demonstrates Python programming concepts including Object-Oriented Programming (OOP), Functions, List and Dictionary Comprehensions, Data Cleaning, Data Analysis, and Data Visualization.

## Features

- Read student data from CSV files
- Calculate Total, Average and Grade
- Find Topper and Lowest Scorer
- Branch-wise Performance Analysis
- Attendance Analysis
- Subject-wise Statistics
- Pass/Fail Analysis
- Merge Multiple DataFrames
- Generate Graphical Reports

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
